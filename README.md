# SimpleSpringCloudQuickStarter

A SimpleSpringCloudQuickStarter Project

一个可以作为项目基石的快速启动的SpringCloud项目。

## 具备的功能

- SpringCloud(✔️)
- SpringGateway(✔️)
- MyBatisPlus(✔️)
- Leaf分布式ID生成(✔️)
- 接口日志打印(❌)
- 权限控制(❌)
- Nacos 注册中心和配置中心(✔️)
- 分库分表(❌)
- Redis(❌)
- MessageQueue(❌)
- 链路追踪(❌)
- ELK(❌)
- 未完待续(❌)
